# cbr-auto-deployment-tool

Installation
1) Install node.js
2) $ npm install --global gulp
3) $ npm install --save-dev gulp
4) $ npm install --save-dev gulp-util

Tasks
1) watch:source
2) deploy:js
3) deploy:jsp
4) deploy:flow
5) deploy:java
6) deploy:bhlibs
7) deploy:integration
8) portlet:compile
9) integration:compile
